# Match Mania - Tile Matching Game

Match Mania is a simple tile matching game built using HTML, CSS, and JavaScript. The objective of the game is to flip tiles and match pairs of tiles with the same images.


## Features

- **Tile Flipping**: Click on tiles to reveal the image underneath.
- **Matching Logic**: Match two tiles with the same to keep them open.
- **Game Over Condition**: Complete the game by matching all pairs of tiles.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## How to Play

1. Clone or download the repository.
2. Open `index.html` in your web browser.
3. Click on tiles to flip them and try to match pairs with the same images.
4. Continue until all tiles are matched to complete the game.
